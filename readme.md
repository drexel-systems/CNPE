# Cloud Native Platform Engineering (CNPE)

This repository provides starter code and course resources for a series of courses that I teach on cloud computing.  Navigate directly to:

1. [CNPE: Undergrad Focus](./undergrad/readme.md) - Build from Scratch (properly)
2. [CNPE: Graduate Focus](./grad/readme.md) - Architectural investigation; Critical evaluation tradeoffs 

Both courses will center around helping a fictitious company called NovaSpark on optomizing thier cloud platform as the company grows. See more about [NovaSpark](./novaspark/readme.md)

Note please bookmark this repo if you are taking this course.  Lectures will be provided via Drexel's Canvas LMS, this repo will be used to distribute lab starter code.
